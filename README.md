## New defaults and built-in mappings

- gra → code actions
- gri → implementations
- grn → rename
- grr → references
- grt → type definition
- grx → run codelens
- gO → document symbols
- Ctrl-S in Insert mode → signature help


## Incremental Selection

- v_an - select parent node
- v_in - select child node
- v_]n - select prev node
- v_[n - select next node
